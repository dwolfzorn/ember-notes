# Taxonomy

**Taxonomy** classifies the biology of an **Adversary** in Crucible. Each Taxonomy defines a variety of basic properties:

- Category (such as whether a creature is an Beast, Celestial, Humanoid, etc.)
- Base size of an adversary (in grid spaces)
- Stride (movement speed in feet)
- Whether the creature can wear typical humanoid equipment or cast spells
- [[Ability Scores]] which determine baseline attributes
- [[Resistances and Vulnerability]] and Immunities which define how damage resistances scale with threat level and whether the creature is immune to any specific types of damage.
- Any talents that all creatures with this taxonomy should start with

#### Example: Hobgoblin Taxonomy

Hobgoblins are humanoids which typically congregate in martial societies which are orderly and disciplined. Hobgoblins are both cunning and uncompromising, making dangerous foes or powerful allies.

The Hobgoblin Taxonomy defines scaling weights such that Strength, Intellect, and Toughness evolve more frequently while Wisdom, Presence, and Dexterity evolve more slowly.

Many different Adversaries will use the same Taxonomy, for example a Level 3 **Hobgoblin Archer**, Level 4 **Hobgoblin Fighter**, and a Level 6 **Hobgoblin Witch Doctor**. Each of these adversaries might have a different [[Archetype]] and therefore have different attributes and talents, but they would all share a common progression from their shared Hobgoblin Taxonomy.
