# Dice Mechanics

Crucible uses the standard set of role-playing game dice; relying upon 4, 6, 8, 10, 12, and 20-sided dice for various rolls. These dice are referred to as **d4**, **d6**, **d8**, **d10**, and **d12**, where a number that precedes the die type represents the quantity of that die to roll, for example **4d6** provides instruction to roll four six-sided dice.

> Crucible is designed specifically for Foundry Virtual Tabletop, but its dice mechanics are possible to play outside of a digital environment, albeit with a fairly high level of bookkeeping required.

When your character performs heroic actions within the world, whether they be feats of martial prowess or clever social maneuvers, the outcomes of the attempted action are discovered by rolling dice.

## Standard Check

The standard approach towards determining the outcome of a dice check in Crucible uses a dice pool comprised of three dice, typically 3d8, which are added together. The exact dice used in the pool are modified depending on the **Boons** or **Banes** which apply to the roll. Added to the result of these three dice are Ability, Skill, and Enchantment bonuses which further increase the value of the result. This type of dice roll is referred to as a **Standard Check**.

Standard Check

=

3d8[Boons,Banes] + Ability Bonus + Skill Bonus + Enchantment Bonus

## Bonus Stacking

Any given roll may only have one source of **Ability Bonus**, **Skill Bonus**, or **Enchantment Bonus**. If certain Actions, Talents, or Equipment provide multiple methods of obtaining a bonus for a roll, only the largest bonus is used.

#### Example: Multiple Enchantment Bonuses

Zarajah is an accomplished spellcaster who has obtained several powerful artifacts. A set of mages robes that provide a +1 Enchantment Bonus to all [[Spellcraft]], and a powerful staff which gives a +2 Enchantment Bonus to spells cast using [[Rune: Flame]]. When casting a spell using the Earth rune, Zarajah has a +1 Enchantment Bonus on any dice roll involved that spell. When casting a spell using the Flame rune, Zarajah has a +2 Enchantment Bonus.
