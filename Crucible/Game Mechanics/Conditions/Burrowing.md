# Burrowing

A creature that is submerged entirely within the ground is **Burrowing**. While Burrowing, a creature cannot see or be seen using light-based perception.

#### Automation

The Burrowing condition is applied automatically when a creature ends its own movement using the **Burrow** movement action, and is removed automatically when that creature later ends a movement using any other movement action.
