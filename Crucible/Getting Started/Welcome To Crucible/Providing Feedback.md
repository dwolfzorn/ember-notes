# Providing Feedback

We welcome feedback on your experience with Crucible. This data helps us identify areas of the game which need improvement, collect bugs that need to be fixed, and prioritize the development of new features.

### Discord

The Foundry Virtual Tabletop community [Discord](https://discord.gg/foundryvtt) server is a primary hub for discussion and feedback. We have a dedicated channel for Crucible feedback, and we encourage users to join the server and participate in the discussion. Please join the discussion in the **#crucible** channel of our Discord server.

### GitHub

Reproducible bug reports and clearly articulated feature requests can be submitted to the [Crucible issue tracker](https://github.com/foundryvtt/crucible/issues) on GitHub.
