# Ancestry

The first step in character creation is selecting an [[Ancestry]], which classifies the biology and physiology of a character in Crucible, and grants the newly created hero specific starting traits.

You can change your character's Ancestry at any point during the character creation, but be aware that doing so may impact your currently assigned Ability Scores if you have already chosen them and that may require re-selecting them before completing your character creation.

While Crucible provides a number of example ancestries, it also allows use of custom ancestries by providing a way for GMs to configure their own compendium packs to use as a source during character creation. To add your compendium as a source, go to **Configure Settings ➤ Crucible ➤ Configure Compendium Sources**.

## Ancestry Features

Each Ancestry includes:

- A **Primary Ability** Score (which begins at 3)
- A **Secondary Ability** Score (which begins at 2)
- Optionally, a **Damage Resistance** (+3) and a **Damage Vulnerability** (-3).
- A **Size** value in grid spaces (typically 4)
- A **Stride** value in grid spaces that determines the starting movement speed (typically 10)
- An **Ancestral Talent** which grants a specific special ability or knowledge to a character of that ancestry.

Currently, the built-in Ancestries which are provided with the Crucible system do not have unique starting talents. Unique talents for these built-in ancestries will be added in a future update.
