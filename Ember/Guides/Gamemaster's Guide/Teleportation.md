# Teleportation

The Gamemaster has the ability to teleport a caravan to any other hex or Location in the world using the ​​ button in the Scene Controls interface element found on the left side of the screen. This teleportation interface can be used to instantly move the party to a different hex or to a specific Location in the game world.

> [!warning] Gamemaster
> #### Designating a Teleportation Target
>
> Targeting a specific hex using the ​​ tool will automatically mark that hex as the desired teleportation destination.
