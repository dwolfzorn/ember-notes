# System Requirements

## Server Requirements

For the server or computer which is hosting the Foundry Virtual Tabletop game to perform well with Ember we recommend the following specifications.

- The [Hosting Requirements](https://foundryvtt.com/article/requirements/) listed on the Foundry Virtual Tabletop website.
- Foundry Virtual Tabletop version `14.358` or greater.
- The Dungeons & Dragons Fifth Edition game system version `5.3.0` or greater.
- OR the Crucible game system version `0.9.0` or greater.
- At least `4GB` of RAM and `5GB` of available disk space on the server which hosts the game.
- An internet upload speed of at least `24mbps (3MB/s)` is recommended because Ember requires transferring large media files to all connected clients.

## Player Requirements

For the gamemaster and players to have a great experience in Ember we require and recommend the following specifications.

- The [Recommended Client Specifications](https://foundryvtt.com/article/requirements/) listed on the Foundry Virtual Tabletop website.

For players with a less powerful computer we recommend reducing the Performance Mode — which is configurable in game settings — to either Medium or Low. Reducing the performance mode will downgrade certain effects in-game resulting in improved performance for low-powered machines.
