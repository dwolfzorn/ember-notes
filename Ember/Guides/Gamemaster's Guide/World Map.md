# World Map

Ember includes a World Map which depicts the major geographical features of the [[World of Ember]]. You can navigate to the world map easily at any time by clicking the ​ icon in the Calendar interface element at the top of the screen.

This World Map is used only for reference, not for gameplay. All gameplay pertaining to the party's in-world location is handled by the [[Gameplay Overview]].

## Using the World Map

The Ember World Map includes prewritten annotations that you can access to learn more about places within the [[World of Ember]]. Place names with associated information in the Setting Compendium will increase in size when you mouseover them — double-clicking these place names will bring you to the relevant Journal entry in the Setting Compendium.

#### Search and Discoverability

In the future, the World Map will include interface elements to search and more easily explore elements of the World Map.
