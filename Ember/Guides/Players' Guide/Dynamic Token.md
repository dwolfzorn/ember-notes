# Dynamic Token

The final step of the Character Creation process involves customizing your Character Token's appearance. This Token can be used as the main visual reference of your character, but more importantly, it represents your character on the [[Area Maps]] you will explore during the game.

When using the Token Maker during the Character Creation process, the choices you made in previous steps automatically determines the set of anatomy and equipment options that are available to you. For example, if you select a particular Ancestry, you will only be able to choose anatomical parts from that Ancestry.

### Anatomy

First, you will choose your Token's anatomy, which determines your character's basic physical features. Anatomy includes body shape, face, eyes, arms, feet, and hands. To change these parts, click on the arrows on either side of the name displayed on the left side.

### Equipment

Secondly, you will choose equipment, which can be selected above the visual Token appearance. This allows you to choose items, objects, and clothing for your character to wear. Once again, to change these parts, click the arrows on either side of the name displayed on the left.

### Colors

Each anatomy or equipment piece can be colored via the small box on the right side of the Character Creation interface. Clicking this box reveals a color palette with hex codes for each color option. For some aspects of your character, you can select multiple colors. If you see options labeled 1 and 2, such as "Skin 1" or "Skin 2," this typically means you can assign different colors to different sides of the same part.

> [!warning] Gamemaster
> #### Random Appearances
>
> There is a small Randomize button with two interlocking arrows next to Equipment. This button randomizes your anatomy and equipment, using pre-selected color choices that make the most sense for the Ember setting.

> [!danger] Hazard
> To configure your Token appearance after Character Creation, click the three small vertical dots in the top right corner of the Character Sheet, then select Ember Dynamic Token.
