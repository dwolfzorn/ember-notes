# Cosmos Map

Ember includes a map of [[Cosmos]] which can be viewed at any time by clicking the ​ icon in the Calendar interface element at the top of the screen.

This visual representation of the Cosmos is accurate to the current world time. As time progresses, so too do the positions of the sun, moons, and inner realms that make up the Cosmos.

Each of the inner realms is depicted around the perimeter of [[The Weave]]. The realm or realms closest to the top of the visualization are those which exert the greatest influence upon Ember itself.

Meanwhile, each moon has a different orbital path depicted on the Cosmos Map. The solid dot at one end of this path represents the position where this moon is closest to Ember and thus exerts the greatest influence.

## Using the Cosmos Map

The visualizations of the orbits of Ember's moons are of particular importance. As a particular moon reaches its zenith, Ember is bathed in the elemental energy associated with that lunar body, which can have strange effects upon items, attuned individuals, and more — you can always check how close a moon's orbit is to apex via the Cosmos Map.
