# Spare Bathroom

> [!warning] Gamemaster
> #### Access to Spare Bathroom
>
> Access to Spare Bathroom is **restricted** at all times. (See [[Gameplay Details]] for more on access levels.)
>
> #### Locating Funar Cevher
>
> See [[Casing the Joint]] for details. If the party uses the[[Locator Rod]] in this area, they find that Funar Cevher is **in the vicinity.** This is the baseline result when using the Locator Rod in and around Marlstone Manor.

> [!quote] Read Aloud
> This bathroom seems as if it hasn't been used in some time. Whatever products were once offered to those using it, the sink now has nothing on it but empty bottles.

> [!tip] Exploration
> #### An Impromptu Container
>
> The empty bath vials in this room can be used as potion containers, serving as [[Vials]].
